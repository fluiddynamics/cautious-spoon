# cautious-spoon

確率統計の問題集を作っています．

## 目的
* 確率統計の計算問題を紹介する
* 解くことで，確率統計の理解が深まる

## 構成
1.  計算問題
2. 数式の問題

## 引用について
一部の問題は書籍に書いてあった問題を引用しています．
その場合は，「どの本の何ページ」を明記しています．

引用したのは次の本などです．

* [まずはこの一冊から　意味がわかる統計解析](https://www.beret.co.jp/books/detail/470)
* [数理統計学（改訂版）](https://www.shokabo.co.jp/mybooks/ISBN978-4-7853-1411-8.htm)
* 大学入試センター試験
